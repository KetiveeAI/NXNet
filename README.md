# NXNet
Native networking stack for the Ketivee ecosystem.

## Components
- nxbase — foundation (string, hashmap, log, timer)
- nxcrypto — TLS, hash, base64
- nxhttp — HTTP/1.1, HTTP/2, connection pool, cache
- networking — DNS, SSL, cookies, WebSocket

## Used In
- Zepra Browser
- ZepraSearch / ZepraBot
- HiiFie
